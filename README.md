# wemanity_test
To run file : 

Prerequisite: 

    -install nodejs if not installed    
    -install Typescript    
        open CMD and run : "npm install -g typescript"
        
Compile ts files:

    In CMD run :
        - "tsc golden-master-text-test.ts"
        
Run test :

    In CMD run :
        - "node golden-master-text-test.js"
